Estrutura condicional If Else
É uma estrutura de controle que permite definir que um certo bloco de comandos somente sera executado dependendo de uma condição

if(<condição){
  <comando1>
  <comando2>
}else {
 <comando1>
 <comando2>
}

Estrutura condicional ternária
Estrutura opcional ao if-else quando se deseja decidir um Valor com base em uma condição
(condição) ? valor_se_verdadeiro : valor_se_falso

(2>4) ? 50:80 > 80

Escopo e Inicialização

Escopo de uma variavel é a região do programa onde a variável é válida, ou seja, onde ela pode ser referenciada.
Uma variavel não pode ser usada se não for iniciada
